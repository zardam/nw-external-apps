6.42.12  Avoir la liste permutée : revlist

revlist a comme argument une liste (resp séquence, chaîne).
revlist renvoie la liste (resp séquence chaîne) dans l’ordre inverse.
On tape :

revlist([0,1,2,3,4])

On obtient :

[4,3,2,1,0]

On tape :

revlist([0,1,2,3,4],3)

On obtient :

3,[0,1,2,3,4]

On tape :

revlist("abcd")

On obtient :

"dcba"
