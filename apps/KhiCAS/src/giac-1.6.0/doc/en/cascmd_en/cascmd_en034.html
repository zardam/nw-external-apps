3.6.4  Printing a session

You can print a session with the File▸Print▸to printer menu item.

If you prefer to save the printed form as a file, you can use the File▸Print▸preview menu item. You will prompted for a file name to save the printed form in; the file will be a PostScript file, so the name should end in .ps.If you only want to save certain levels in printable form, you can use the File▸Print▸preview selected levels menu item; this file will be encapsulated PostScript, so the name should end in .eps.
